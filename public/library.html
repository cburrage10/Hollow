Browser Extension

The browser extension allows you to capture text from Kindle Cloud Reader and any webpage.

Chrome Installation

  1. Download the extension folder from the project
  2. Open Chrome and go to chrome://extensions
  3. Enable "Developer mode" (toggle in top right)
  4. Click "Load unpacked" and select the extension folder
  5. The extension icon will appear in your toolbar

Safari Installation (iOS/macOS)

  1. Safari Web Extensions require Xcode to package
  2. Open the Safari extension project in Xcode
  3. Build and run to install on your device
  4. Enable in Safari Settings > Extensions
